## Em desenvolvimento!

Library PHP pra integrar com o [Tray Checkout](http://traycheckout.com.br/).

### Instalação

Adiciona a linha abaixo no seu `composer.json` e execute um `composer update`.

```javascript
    "require": {
        "escapework/tray-php": "0.1.*"
    }
```
